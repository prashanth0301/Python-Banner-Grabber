# Banner Grabber

A simple Python project that connects to a TCP service and retrieves its banner.

## Features
- Banner grabbing
- HTTP GET support
- Exception handling
- Input validation

## Run
```bash
python banner_grabber.py
```

## Test
- Host: scanme.nmap.org  Port: 22
- Host: scanme.nmap.org  Port: 80

Use only on systems you own or are authorized to test.
